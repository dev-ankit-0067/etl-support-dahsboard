Prompt Title
Design a Production Support & Cost Insights Web Dashboard for AWS ETL Pipelines using a light, professional enterprise theme, optimized for executive leadership and production support teams.

Design Style Guidelines

Theme: Light color palette (white, light gray backgrounds)
Primary Accents: Soft blue, teal, muted green
Status Colors:

Red → Action required
Amber → Risk / Near breach
Green → Healthy


Typography: Clean enterprise fonts (Inter / Roboto / SF Pro–style)
Visual Language: KPI cards, clean tables, minimal icons, subtle shadows
Avoid: Dark themes, excessive gradients, decorative charts
